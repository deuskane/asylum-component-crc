# CRC
CSR for CRC

| Address | Registers |
|---------|-----------|
|0x0|data0|
|0x1|data1|
|0x2|crc0|
|0x3|crc1|

## 0x0 data0
Data byte0 - write start crc

### [7:0] value
Data Byte 0

## 0x1 data1
Data byte1

### [7:0] value
Data Byte 1

## 0x2 crc0
CRC value byte 0

### [7:0] value
CRC Byte 0

## 0x3 crc1
CRC value byte 1

### [7:0] value
CRC Byte 1
